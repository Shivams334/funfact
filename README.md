# Funfact
This is my first python package for test.


Installation
------------

```Bash
    $ pip install funfact
```
or download, uncompress and install pygrok from [here](https://pypi.python.org/pypi/funfact):

```Bash
    $ tar zxvf funfact-01.tar.gz
    $ cd funfact-01
    $ sudo python setup.py install
```
Usage
---------------
```Python
import funfact
print funfact.fact()
```
